# How to Modify the Website Using GitHub

This guide explains how to make changes to the website using the GitHub website interface. Follow these steps carefully.

---

## 1. Open the Repository on GitHub

1. Go to the GitHub website: [https://github.com](https://github.com).
2. Log in to your account.
3. Navigate to the repository for the website. For example:
   - `YiruCai.github.io`

---

## 2. Locate the File to Edit

1. In the repository, click on the file you want to edit. Common files include:
   - `\_pages\index.html` (for the homepage)
   - `style.css` (for website styling) (Don't need to change mostly)
2. Use the search bar or navigate through the folder structure to find the file.

---

## 3. Edit the File

1. Click the pencil icon (✏️) in the top-right corner of the file to open the editor.
2. Make your changes:
   - **To update text:** Find the text you want to change and replace it with the new text.
   - **To change an image:** Update the image file in the `images` folder and ensure the file name matches the one in the code.
   - **To add a new section:** Copy the structure of an existing section and modify it with your new content.

---

## 4. Save Your Changes

1. Scroll down to the "Commit changes" section.
2. Add a short description of your changes in the "Commit message" box.
3. Click the "Commit changes" button.

---

## 5. Preview Your Changes

1. Wait a few minutes for the changes to be published.
2. Visit the website to ensure your updates are visible.

---

## 6. Need Help?

If you're unsure about something, ask someone familiar with GitHub or web development for assistance.
